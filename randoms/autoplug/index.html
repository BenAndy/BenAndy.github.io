INSTALL

To install, simply drag the above image into your bookmarks bar or folder!

OR

Add a bookmark with the address "javascript:(function(){$.getScript('https://benandy.github.io/js/autoplug.js');}());".

STARTING AUTOPLUG

When you're in a Plug.dj community, simply click the bookmark! AutoPlug will start doing it's thing. You will see it log a few things in the chat - This is how you know it's started. The chat box will be flashing with messages this is how you know AutoPlug is still running.

STOPPING AUTOPLUG

To stop AutoPlug, simply reload the page (F5)! Don't worry - You won't lose your spot in the wait list.
